Lito's Chefs Knife (object)

Attributes:
knife_type = "chefs"
is_sharp = true
knife_length = 9.5
owner_name = "Lito"
is_dirty = false


Methods:
chop: onions are chopped. Attribute is_sharp is now false and is_dirty is now true.
change_owner: No. No change of ownership
be_cleaned: The knife is cleaned at the end of the day. Attribute is_dirty is now false.
be_sharpened: The knife is sharpened. Attribute is_sharp is now true.
